Many of these papers are also available on:

- Google Scholar
- Authors' personal websites
- Conference proceedings

## Note on Authentication

Direct programmatic download from Hugging Face Papers requires authentication. To download papers:
1. Visit the paper page using the links above
2. Click the download button on the page
3. Save the PDF to this directory with a descriptive filename
